1003 我要通过！ (20 分)

作者: Milk 's Home 日期: 2020-03-29 阅读时间:2.9 分钟 字数:705 分类: 代码练习 阅读量: -

题目

“答案正确”是自动判题系统给出的最令人欢喜的回复。本题属于 PAT 的“答案正确”大派送 —— 只要读入的字符串满足下列条件，系统就输出“答案正确”，否则输出“答案错误”。

得到“答案正确”的条件是：

  1. 字符串中必须仅有 P、 A、 T这三种字符，不可以包含其它字符；
  2. 任意形如 xPATx 的字符串都可以获得“答案正确”，其中 x 或者是空字符串，或者是仅由字母 A 组成的字符串；
  3. 如果 aPbTc 是正确的，那么 aPbATca 也是正确的，其中 a、 b、 c 均或者是空字符串，或者是仅由字母 A 组成的字符串。

现在就请你为 PAT 写一个自动裁判程序，判定哪些字符串是可以获得“答案正确”的。

输入格式：

每个测试输入包含 1 个测试用例。第 1 行给出一个正整数 n (<10)，是需要检测的字符串个数。接下来每个字符串占一行，字符串长度不超过 100，且不包含空格。

输出格式：

每个字符串的检测结果占一行，如果该字符串可以获得“答案正确”，则输出 YES，否则输出 NO。

输入样例：

8
PAT
PAAT
AAPATAA
AAPAATAAAA
xPATx
PT
Whatever
APAAATAA

输出样例：

YES
YES
YES
YES
NO
NO
NO
NO

思路总结：

题目中的三点要求，其中最后两条有点难以理解

任意形如 xPATx 的字符串都可以获得“答案正确”，其中 x 或者是空字符串，或者是仅由字母 A 组成的字符串；

从这个条件可以知道字符串"P"和"T"之间至少存在一个“A”

如果 aPbTc 是正确的，那么 aPbATca 也是正确的，其中 a、 b、 c 均或者是空字符串，或者是仅由字母 A 组成的字符串。

将这个条件与上一个条件结合使用，可以看出来在字符串"P"和"T"之间每增加一个“A”时，需要在字符串后面追加一个字符串a，也就是说中间的字符长度与"T"之后的字符长度是线性关系

得到等式： len(a) * len(b) == len(c)

于是代码如下：

#include<iostream>
#include<map>

using namespace std;

int main(){
    int n;
    cin >> n;
    for(int i=0;i < n;i++){
        int p,t;
        string s;
        cin >> s;
        map<char,int> m;
        for(int j=0;j < s.size();j++){
            m[s[j]]++;
            if(s[j] == 'P') p=j;
            if(s[j] == 'T') t=j;
        }
        if(m['P']==1 && m['T']==1 && m['A']!=0 && t-p!=1 && 
           m.size()==3 && p*(t-p-1)==s.size()-t-1)
        {
            cout << "YES\n";
        }else{
            cout << "NO\n";
        }
    }
    return 0;
}
