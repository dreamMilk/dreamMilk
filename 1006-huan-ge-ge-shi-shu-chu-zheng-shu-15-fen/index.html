1006 换个格式输出整数 (15分)

作者: Milk 's Home 日期: 2020-04-03 阅读时间:2.1 分钟 字数:500 分类: 代码练习 阅读量: -

题目

让我们用字母 B 来表示“百”、字母 S 表示“十”，用 12...n 来表示不为零的个位数字 n（<10），换个格式来输出任一个不超过 3 位的正整数。例如 234 应该被输出为 BBSSS1234，因为它有 2 个“百”、3 个“十”、以及个位的 4。

输入格式：

每个测试输入包含 1 个测试用例，给出正整数 n（<1000）。

输出格式：

每个测试用例的输出占一行，用规定的格式输出 n。

输入样例1：

234

输出样例1：

BBSSS1234

输入样例2：

23

输出样例2：

SS123

思路：

题目与1004 成绩排名 (20分)的难度一样属于比较简单的题目，只是原本输出每个位数的题目进行了变化

起手是想按照原来的套路进行修改即可，发现输出时需要从高位数开始，如果从低位数开始取位数必定需要一个变量来保存这些数，然后进行判断循环输出得到正确输出

上面的想法对于空间利用有点奢侈，但是上手起来也挺顺手，不过为了能省则省的原则，进行一下改变

从高位开始取位数，然后立即判断输出，避免记忆这些临时数据

因为习惯将复用的步骤抽离出来写成函数，看着其实也挺清爽

如果是追求代码精炼可以尝试自行改动

于是代码如下：

#include<iostream>

using namespace std;

int print(int i, int key){
    switch (key){
    case 0:
        for(int j=0;j<i;j++){
            cout << j+1;
        }
        break;
    case 1:
        for(int j=0;j<i;j++){
            cout << 'S';
        }
        break;
    case 2:
        for(int j=0;j<i;j++){
            cout << 'B';
        }
        break;
    }
    return 0;
}

int main(){
    int num;
    int i;
    cin >> num;
    if(num > 99){
        i = num/100;
        print(i,2);
        num = num-i*100;
    }
    if(num > 9){
        i = num/10;
        print(i,1);
        num = num-i*10;
    }
    if(num > 0){
        i = num;
        print(i,0);
    }
    return 0;
}
