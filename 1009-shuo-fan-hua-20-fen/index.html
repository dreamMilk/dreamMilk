1009 说反话 (20分)

· 2020-04-06 · # 代码练习

题目

给定一句英语，要求你编写程序，将句中所有单词的顺序颠倒输出。

输入格式：

测试输入包含一个测试用例，在一行内给出总长度不超过 80 的字符串。字符串由若干单词和若干空格组成，其中单词是由英文字母（大小写有区分）组成的字符串，单词之间用 1 个空格分开，输入保证句子末尾没有多余的空格。

输出格式：

每个测试用例的输出占一行，输出倒序后的句子。

输入样例：

Hello World Here I Come

输出样例：

Come I Here World Hello

思路：

题目看着十分简单，但是还是不能直击要害

题目核心是对栈的使用，（数据结构让猪学了）只需将每个单词进行入栈，然后出栈即可得到结果

另外补充一些奇怪的知识🧠：

本题中的输入控制while(cin >> s)，因为第一次看见，不知道这个如何运行结束

于是查阅到 end of file简称EOF，不是文件结尾就继续进行读取，在OJ系统中会自动实现结束输入，而在自己调试运行时却需要手动实现，在cmd控制台输入crtl + Z或crtl + D即可结束输入

如果实在无法理解：可以移步百度，搜索关键字ctrl + d手动结束输入即可找到图解我实在不想贴图

于是代码如下：

#include<iostream>
#include<stack>

using namespace std;

int main() {
    stack<string> v;
    string s;
    while(cin >> s){
        v.push(s);
    }
    cout << v.top();
    v.pop();
    while(!v.empty()){
        cout << " " << v.top();
        v.pop();
    }
    return 0;
}

总结

关于数据结构的短期使用感想：

  • map键值对：主要用于两种不同数据类型时，比如字符的计数等
  • set排序：主要用于对数据的排序有要求时，比如成绩排名等
  • vector数组：更多是像一个不定长的数组使用，但是不容易定位元素
  • stack栈：当需要对元素逆序时，可以考虑使用栈
