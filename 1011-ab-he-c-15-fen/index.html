1011 A+B 和 C (15分)

· 2020-04-08 · # 代码练习

题目

给定区间 [−2312^{31}231,2312^{31}231] 内的 3 个整数 A、B 和 C，请判断 A+B 是否大于 C。

输入格式：

输入第 1 行给出正整数 T (≤10)，是测试用例的个数。随后给出 T 组测试用例，每组占一行，顺序给出 A、B 和 C。整数间以空格分隔。

输出格式：

对每组测试用例，在一行中输出 Case #X: true 如果 A+B>C，否则输出 Case #X: false，其中 X 是测试用例的编号（从 1 开始）。

输入样例：

4
1 2 3
2 3 4
2147483647 0 2147483646
0 -2147483648 -2147483647

输出样例：

Case #1: false
Case #2: true
Case #3: true
Case #4: false

思路：

题目与上一题相似，普通的数学问题

唯一需要注意的是区间 [−2312^{31}231,2312^{31}231]

INT只能表示[−2312^{31}231,231−12^{31}-1231−1] ，这个涉及到计组的数据补码存储的知识

首先INT是32位，有符号的INT首位是符号位，其余为数据位，相反无符号的unsigned int全是数据位

我们主要说有符号的INT，从位数上看好像剩余31位应该对应231−12^{31}-1231−1，却发现在负数边多了一个

这是因为表示0的时候发现有两种形式，一种是+0，一种是-0，于是对于-0进行变化表示为2312^{31}231，而不是-0

以上是对补码的一个胡乱解释（小白向）并不专业，如果感兴趣可以查阅专业的书籍进行学习📚

对bool类型输出时，添加boolalpha可以直接输出true和false，否则输出0和1

于是代码如下：

#include<iostream>

using namespace std;

int main() {
    int t;
    long int a,b,c;
    cin >> t;
    for(int i=0;i<t;i++){
        cin >> a >> b >> c;
        cout << "Case #" << i+1 << ": " << boolalpha << (a+b>c) << endl;
    }
    return 0;
}
