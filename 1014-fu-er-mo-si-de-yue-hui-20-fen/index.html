1014 福尔摩斯的约会 (20分)

作者: Milk 's Home 日期: 2020-04-12 阅读时间:3.7 分钟 字数:834 分类: 代码练习 阅读量: -

题目

大侦探福尔摩斯接到一张奇怪的字条：我们约会吧！ 3485djDkxh4hhGE 2984akDfkkkkggEdsb s&hgsfdk d&Hyscvnm。大侦探很快就明白了，字条上奇怪的乱码实际上就是约会的时间星期四 14:04，因为前面两字符串中第 1 对相同的大写英文字母（大小写有区分）是第 4 个字母 D，代表星期四；第 2 对相同的字符是 E ，那是第 5 个英文字母，代表一天里的第 14 个钟头（于是一天的 0 点到 23 点由数字 0 到 9、以及大写字母 A 到 N 表示）；后面两字符串第 1 对相同的英文字母 s 出现在第 4 个位置（从 0 开始计数）上，代表第 4 分钟。现给定两对字符串，请帮助福尔摩斯解码得到约会的时间。

输入格式：

输入在 4 行中分别给出 4 个非空、不包含空格、且长度不超过 60 的字符串。

输出格式：

在一行中输出约会的时间，格式为 DAY HH:MM，其中 DAY 是某星期的 3 字符缩写，即 MON 表示星期一，TUE 表示星期二，WED 表示星期三，THU 表示星期四，FRI 表示星期五，SAT 表示星期六，SUN 表示星期日。题目输入保证每个测试存在唯一解。

输入样例：

3485djDkxh4hhGE 
2984akDfkkkkggEdsb 
s&hgsfdk 
d&Hyscvnm

输出样例：

THU 14:04

思路：

题目主要是涉及字符串比对问题，总结一下需要注意的地方：

一个星期只有七天，所以第一个需要比对的字符范围应该是A~G；小时的范围是0~9和A~N；分钟的范围是所有大小写字母

其他的细节题目已经大致给出

关于比对字符串时的细节，如果字符串中含有多个满足要求的字符，以前两个为准

原本这个没有在我的考虑之类，只想着避免多次循环浪费时间，在得到所需的结果后直接跳出循环，看来以后读题时需要多注意一些问题细节，可以避免提交时存在部分测试点无法通过的情况💡

于是代码如下：

#include<iostream>

using namespace std;

int print(char c) {
    switch (c) {
    case 'A':
        cout << "MON ";
        break;
    case 'B':
        cout << "TUE ";
        break;
    case 'C':
        cout << "WED ";
        break;
    case 'D':
        cout << "THU ";
        break;
    case 'E':
        cout << "FRI ";
        break;
    case 'F':
        cout << "SAT ";
        break;
    case 'G':
        cout << "SUN ";
        break;
    }
}

int main() {
    string s1,s2,s3,s4;
    int index=0;
    bool flag = true;
    cin >> s1 >> s2 >> s3 >> s4;
    for(int i=0; i<s1.length() && i<s2.length(); i++) {
        if(!flag && s1[i]>='0' && s1[i]<='9' && s1[i]==s2[i]) {
            cout << '0' << s1[i] << ':';
            break;
        }
        if(!flag && s1[i]>='A' && s1[i]<='N' && s1[i]==s2[i]) {
            cout << s1[i]-'A' + 10 << ':';
            break;
        }
        if(s1[i]>='A' && s1[i]<='G' && s1[i]==s2[i] && flag) {
            print(s1[i]);
            flag = false;
        }
    }
    for(int i=0; i<s3.length() && i<s4.length(); i++) {
        if(s3[i]>='A' && s3[i]<='z' && s3[i]==s4[i]) {
            if(i < 10) cout << '0' << i;
            else cout << i;
            flag = true;
        }
        if(flag) break;
    }
    return 0;
}
