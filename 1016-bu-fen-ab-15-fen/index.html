1016 部分A+B (15分)

· 2020-04-14 · # 代码练习

题目

正整数 A 的“DAD_ADA​（为 1 位整数）部分”定义为由 A 中所有 DAD_ADA​ 组成的新整数 PAP_APA​。例如：给定 A=3862767，DAD_ADA​=6，则 A 的“6 部分”PAP_APA​ 是 66，因为 A 中有 2 个 6。

现给定 A、DAD_ADA​、B、DBD_BDB​，请编写程序计算 PAP_APA​+PBP_BPB​。

输入格式：

输入在一行中依次给出 A、DAD_ADA​、B、DBD_BDB​，中间以空格分隔，其中 0<A,B<101010^{10}1010。

输出格式：

在一行中输出 PAP_APA​+PBP_BPB​ 的值。

输入样例1：

3862767 6 13530293 3

输出样例1：

399

输入样例2：

3862767 1 13530293 8

输出样例2：

0

思路：

题目描述的逻辑十分清楚，简单捋一下：

DAD_ADA​、DBD_BDB​：可以使用int类型，也可以使用char类型，只需计算时依靠‘0’进行转换比较

在计算PAP_APA​、PBP_BPB​时，使用P = P*10+D公式可以避免设置额外的临时变量

于是代码如下：

#include<iostream>

using namespace std;

int main() {
    string s1,s2;
    char n1,n2;
    int num1=0,num2=0;
    cin >> s1 >> n1 >> s2 >> n2;
    for(int i=0;i<s1.length();i++){
        if(s1[i]==n1) num1 = num1*10+(n1-'0');
    }
    for(int i=0;i<s2.length();i++){
        if(s2[i]==n2) num2 = num2*10+(n2-'0');
    }
    cout << num1 + num2 << endl;
    return 0;
}
