1017 A除以B (20分)

· 2020-04-15 · # 代码练习

题目

本题要求计算 A/B，其中 A 是不超过 1000 位的正整数，B 是 1 位正整数。你需要输出商数 Q 和余数 R，使得 A=B×Q+R 成立。

输入格式：

输入在一行中依次给出 A 和 B，中间以 1 空格分隔。

输出格式：

在一行中依次输出 Q 和 R，中间以 1 空格分隔。

输入样例：

123456789050987654321 7

输出样例：

17636684150141093474 3

思路：

题目是求商和余数：

被除数是不超过 1000 位的正整数，因此使用字符串进行操作

除数是1 位正整数，推荐使用int类型，因为char类型操作需要额外转换

本次的代码逻辑考虑欠佳，有需要改进的地方

首位数字小于除数时的处理应放置在循环外，避免反复进行无意义的判断

所得商应该及时输出，避免额外开辟空间进行存储

可能在更多的细节方面也需要改进，日后再重新整理本题🗓

于是代码如下：

#include<iostream>

using namespace std;

int main() {
    string a,q;
    int b;
    int r=0;
    cin >> a >> b;
    for(int i=0;i<a.length();i++){
        int temp = r*10+a[i]-'0';
        if(i == 0 && temp<b){
            r = a[0]-'0';
        }
        else if(temp>=b){
            q += (temp/b) + '0';
            r = temp%b;
        }else{
            q += '0';
            r = a[i]-'0';
        }
    }
    if(q.length() == 0) q='0';
    cout << q << " " << r << endl;
    return 0;
}
