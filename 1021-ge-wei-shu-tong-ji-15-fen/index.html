1021 个位数统计 (15分)

题目

给定一个 k 位整数 N=dk−110k−1d_{k-1}10^{k−1}dk−1​10k−1+⋯+d1101d_110^1d1​101+d0d_0d0​ (0≤did_idi​≤9, i=0,⋯,k−1, dkd_kdk​−1>0)，请编写程序统计每种不同的个位数字出现的次数。例如：给定 N=100311，则有 2 个 0，3 个 1，和 1 个 3。

输入格式：

每个输入包含 1 个测试用例，即一个不超过 1000 位的正整数 N。

输出格式：

对 N 中每一种不同的个位数字，以 D:M 的格式在一行中输出该位数字 D 及其在 N 中出现的次数 M。要求按 D 的升序输出。

输入样例：

100311

输出样例：

0:2
1:3
3:1

思路：

题目不是很难，读完题目应该能反应到map结构：

对于统计数字来说使用键值对十分便捷，此外map结构自带键排序，因此本题只需将字符串使用map存储，然后输出即可

于是代码如下：

#include<iostream>
#include<map>

using namespace std;

int main() {
    string n;
    cin >> n;
    map<char,int> m;
    for(int i=0;i<n.size();i++){
        m[n[i]] ++;
    }
    for(auto i=m.begin();i!=m.end();i++){
        cout << i->first << ":" << i->second << endl;
    }
    return 0;
}
