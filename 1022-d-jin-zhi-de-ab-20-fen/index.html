1022 D进制的A+B (20分)

作者: Milk 's Home 日期: 2020-04-20 阅读时间:1.1 分钟 字数:240 分类: 代码练习 阅读量: -

题目

输入两个非负 10 进制整数 A 和 B (≤ 230−12^{30}-1230−1)，输出 A+B 的 D (1<D ≤10)进制数。

输入格式：

输入在一行中依次给出 3 个整数 A、B 和 D。

输出格式：

输出 A+B 的 D 进制数。

输入样例：

123 456 8

输出样例：

1103

思路：

题目主要是进制转换的问题：

说到十进制转二进制，就能想到"除2取余，逆序排列"法

其他进制的思想类似，也是"除X取余，逆序排列"

既然需要逆序排列了，那就使用栈进行辅助操作，最后全部弹栈即可

于是代码如下：

#include<iostream>
#include<stack>

using namespace std;

int main() {
    int a,b,d;
    cin >> a >> b >> d;
    int t=a+b;
    stack<int> s;
    while(t!=0){
        s.push(t%d);
        t /= d;
    }
    if(s.empty()) cout << "0";
    while(!s.empty()){
        cout << s.top();
        s.pop();
    }
    return 0;
}
