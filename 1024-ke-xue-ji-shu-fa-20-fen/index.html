1024 科学计数法 (20分)

· 2020-04-22 · # 代码练习

题目

科学计数法是科学家用来表示很大或很小的数字的一种方便的方法，其满足正则表达式 [+-][1-9].[0-9]+E[+-][0-9]+，即数字的整数部分只有 1 位，小数部分至少有 1 位，该数字及其指数部分的正负号即使对正数也必定明确给出。

现以科学计数法的格式给出实数 A，请编写程序按普通数字表示法输出 A，并保证所有有效位都被保留。

输入格式：

每个输入包含 1 个测试用例，即一个以科学计数法表示的实数 A。该数字的存储长度不超过 9999 字节，且其指数的绝对值不超过 9999。

输出格式：

对每个测试用例，在一行中按普通数字表示法输出 A，并保证所有有效位都被保留，包括末尾的 0。

输入样例1：

+1.23400E-03

输出样例1：

0.00123400

输入样例2：

-1.2E+10

输出样例2：

-12000000000

思路：

题目只需要注意各种临界条件即可：

先将数字从E分为两部分（忽略符号和小数点），一个是数字部分，一个是指数部分

后面需要对指数部分进行判断操作：

如果指数为负数，则在数字部分前添加0，并在第一个0后面添加.小数点

如果指数大于数字长度（即小数点后移位数比数字位数多），则在数字部分后面添加0

如果指数介于两者之间，则在对应的数字后添加.小数点即可

注意：添加符号时，如果为负数则需要添加符号；如果为正数则不用添加符号

于是代码如下：

#include<iostream>

using namespace std;

int main() {
    string a,s,t;
    cin >> a;
    int i=1;
    while(a[i]!='E') i++;
    s=a[1]+a.substr(3,i-3);
    int n=stoi(a.substr(i+1));
    if(a[0]=='-') cout << '-';
    if(n<0){
        cout << "0.";
        for(int i=-1;i>n;i--) cout << "0";
        cout << s;
    }else if(n>=0){
        if(n < s.length()-1){
            cout << s.substr(0,n+1) << "." << s.substr(n+1);
        }else{
            s.insert(s.length(),n-s.length()+1,'0');
            cout << s;
        }
    }
    return 0;
}
