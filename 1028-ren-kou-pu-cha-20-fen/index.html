1028 人口普查 (20分)

· 2020-04-26 · # 代码练习

题目

某城镇进行人口普查，得到了全体居民的生日。现请你写个程序，找出镇上最年长和最年轻的人。

这里确保每个输入的日期都是合法的，但不一定是合理的——假设已知镇上没有超过 200 岁的老人，而今天是 2014 年 9 月 6 日，所以超过 200 岁的生日和未出生的生日都是不合理的，应该被过滤掉。

输入格式：

输入在第一行给出正整数 N，取值在(0,105]；随后 N 行，每行给出 1 个人的姓名（由不超过 5 个英文字母组成的字符串）、以及按 yyyy/mm/dd（即年/月/日）格式给出的生日。题目保证最年长和最年轻的人没有并列。

输出格式：

在一行中顺序输出有效生日的个数、最年长人和最年轻人的姓名，其间以空格分隔。

输入样例：

5
John 2001/05/12
Tom 1814/09/06
Ann 2121/01/30
James 1814/09/05
Steve 1967/11/20

输出样例：

3 Tom John

思路：

题目只需要注意时间的对比：

题目将输入的时间统一格式，因此在进行时间比较时可以直接使用字符串比较大小

后续的思路基本与输入一组数字寻找最大最小值的题目类似：

如果时间合理，计数加一

如果比最大值大，更新最大值；如果比最小值小，更新最小值

注意：如果没有符合条件的时间，则只需输出有效数即可

于是代码如下：

#include<iostream>

using namespace std;

int main() {
    int n,cont=0;
    string name,birth;
    cin >> n;
    string max_name,max_birth="1814/09/06",min_name,min_birth="2014/09/06";
    for(int i=0;i<n;i++){
        cin >> name >> birth;
        if(birth >= "1814/09/06" && birth <= "2014/09/06"){
            if(birth <= min_birth){
                min_birth = birth;
                min_name = name;
            }
            if(birth >= max_birth){
                max_birth = birth;
                max_name = name;
            }
            cont ++;
        }
    }
    cout << cont;
    if(cont != 0) cout << " " << min_name << " " << max_name;
    return 0;
}
