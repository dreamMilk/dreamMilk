1032 挖掘机技术哪家强 (20分)

作者: Milk 's Home 日期: 2020-04-30 阅读时间:2.0 分钟 字数:495 分类: 代码练习 阅读量: -

题目

为了用事实说明挖掘机技术到底哪家强，PAT 组织了一场挖掘机技能大赛。现请你根据比赛结果统计出技术最强的那个学校。

输入格式：

输入在第 1 行给出不超过 10510^5105 的正整数 N，即参赛人数。随后 N 行，每行给出一位参赛者的信息和成绩，包括其所代表的学校的编号（从 1 开始连续编号）、及其比赛成绩（百分制），中间以空格分隔。

输出格式：

在一行中给出总得分最高的学校的编号、及其总分，中间以空格分隔。题目保证答案唯一，没有并列。

输入样例：

6
3 65
2 80
1 100
2 70
3 40
3 0

输出样例：

2 150

思路：

题目本身基本可以秒杀，但是说一下亮点：

题目中其所代表的学校的编号（从 1 开始连续编号）这个条件的意思比较隐蔽

从这个条件可以知道所有学校的编号不会超过输入的参赛人数N

有了这个条件，可以使用数组进行存储，避免使用map等结构进行存储，提高效率

其实可以稍微处理一下，避免第一个空间被浪费，但是为了代码简单明了就忽略了

通过近些天的题目做一些小结：

在存在一一映射的关系时，可以优先考虑一下使用数组，数组在后续进行排序以及搜索等操作时，会更加方便

于是代码如下：

#include<iostream>
#include<vector>

using namespace std;

int main() {
    int n;
    cin >> n;
    vector<int> v(n+1);
    for(int i=0;i<n;i++){
        int school,score;
        cin >> school >> score;
        v[school] += score;
    }
    int max_score=v[1],max_school=1;
    for(int i=2;i<n;i++){
        if(v[i] > max_score){
            max_score=v[i];
            max_school=i;
        }
    }
    cout << max_school << " " << max_score;
    return 0;
}
