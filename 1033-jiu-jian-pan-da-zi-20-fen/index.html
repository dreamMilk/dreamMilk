1033 旧键盘打字 (20分)

2020-05-01 Views 代码练习399字2 min read繁

题目

旧键盘上坏了几个键，于是在敲一段文字的时候，对应的字符就不会出现。现在给出应该输入的一段文字、以及坏掉的那些键，打出的结果文字会是怎样？

输入格式：

输入在 2 行中分别给出坏掉的那些键、以及应该输入的文字。其中对应英文字母的坏键以大写给出；每段文字是不超过 10510^5105 个字符的串。可用的字符包括字母 [a-z, A-Z]、数字 0-9、以及下划线 _（代表空格）、,、.、-、+（代表上档键）。题目保证第 2 行输入的文字串非空。

注意：如果上档键坏掉了，那么大写的英文字母无法被打出。

输出格式：

在一行中输出能够被打出的结果文字。如果没有一个字符能被打出，则输出空行。

输入样例：

7+IE.
7_This_is_a_test.

输出样例：

_hs_s_a_tst

思路：

题目是对1029 旧键盘的延申：

只需要注意坏掉的键中是否存在➕，如果存在则遇到大写字母跳过输出，反之照常判断

注意使用getline函数，否则会存在测试点无法通过（应该是输入坏键的时候悄悄地加了空格）

于是代码如下：

#include<iostream>
#include<cctype>

using namespace std;

int main() {
    string bad,input;
    getline(cin,bad);
    getline(cin,input);
    bool flag=true;
    if(bad.find('+') != string::npos) flag=false;
    for(int i=0;i<input.length();i++){
        if(bad.find(toupper(input[i])) == string::npos){
            if(isupper(input[i]) && !flag) continue;
            else cout << input[i];
        }
    }
    return 0;
}
