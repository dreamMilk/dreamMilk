1034 有理数四则运算 (20分)

题目

本题要求编写程序，计算 2 个有理数的和、差、积、商。

输入格式：

输入在一行中按照 a1/b1 a2/b2 的格式给出两个分数形式的有理数，其中分子和分母全是整型范围内的整数，负号只可能出现在分子前，分母不为 0。

输出格式：

分别在 4 行中按照 有理数1 运算符 有理数2 = 结果 的格式顺序输出 2 个有理数的和、差、积、商。注意输出的每个有理数必须是该有理数的最简形式 k a/b，其中 k 是整数部分，a/b 是最简分数部分；若为负数，则须加括号；若除法分母为 0，则输出 Inf。题目保证正确的输出中没有超过整型范围的整数。

输入样例：

2/3 -4/2

输出样例：

2/3 + (-2) = (-1 1/3)
2/3 - (-2) = 2 2/3
2/3 * (-2) = (-1 1/3)
2/3 / (-2) = (-1/3)

输入样例：

5/3 0/6

输出样例：

1 2/3 + 0 = 1 2/3
1 2/3 - 0 = 1 2/3
1 2/3 * 0 = 0
1 2/3 / 0 = Inf

思路：

题目主要难点在于给出分子与分母后输出最简分数：

1.判断传入的分子或分母是否为0，如果分子为0，输出0；分母为0，输出Inf

2.判断分子与分母是否异号，如果异号需要按(-k a/b)输出

3.去掉分子分母符号，找出两者的最小公约数，进行约分

4.判断分数是否为整数，如果是，直接输出该整数

5.通过分子分母相除可以得到K，分子与分母取余可得分子，分母不变

还有一些不是重点，却不得不说的：

由于程序过程中涉及到数字相乘，范围可能需要long long int进行存储

求两个数的最小公约数，可以使用辗转取余法，具体数学过程可以百度一下

Code Blocks使用%lld会有警告，但结果正确；使用%l64d运行结果会出现错误，但是没有警告

这个题目应该还是多盘一会，逻辑思路难以理清，这也是目前乙级题目里面有点长度的代码😸

于是代码如下：

#include<iostream>
#include<cmath>

using namespace std;

int divisor(long long a,long long b){
    return a%b==0?b:divisor(b,a%b);
}

int conversion(long long a,long long b){
    if(b==0){
        cout << "Inf";
        return 0;
    }
    bool flag=((a<0&&b>0) || (a>0&&b<0));
    a = abs(a);b = abs(b);
    long long d=divisor(a,b);
    a /= d;b /= d;
    if(flag) cout << "(-";
    if(a%b==0) cout << a/b;
    else if(a>b) cout << a/b << ' ' << a-a/b*b << '/' << b;
    else cout << a << '/' << b;
    if(flag) cout << ')';
    return 0;
}

int main() {
    long long a1,b1,a2,b2;
    scanf("%lld/%lld %lld/%lld", &a1, &b1, &a2, &b2);
    conversion(a1,b1); cout << " + "; conversion(a2,b2); cout << " = "; conversion(a1*b2+a2*b1,b1*b2); cout << endl;
    conversion(a1,b1); cout << " - "; conversion(a2,b2); cout << " = "; conversion(a1*b2-a2*b1,b1*b2); cout << endl;
    conversion(a1,b1); cout << " * "; conversion(a2,b2); cout << " = "; conversion(a1*a2,b1*b2); cout << endl;
    conversion(a1,b1); cout << " / "; conversion(a2,b2); cout << " = "; conversion(a1*b2,b1*a2); cout << endl;
    return 0;
}
