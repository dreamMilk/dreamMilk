1036 跟奥巴马一起编程 (15分)

· 2020-05-04 · # 代码练习

题目

美国总统奥巴马不仅呼吁所有人都学习编程，甚至以身作则编写代码，成为美国历史上首位编写计算机代码的总统。2014 年底，为庆祝“计算机科学教育周”正式启动，奥巴马编写了很简单的计算机代码：在屏幕上画一个正方形。现在你也跟他一起画吧！

输入格式：

输入在一行中给出正方形边长 N（3≤N≤20）和组成正方形边的某种字符 C，间隔一个空格。

输出格式：

输出由给定字符 C 画出的正方形。但是注意到行间距比列间距大，所以为了让结果看上去更像正方形，我们输出的行数实际上是列数的 50%（四舍五入取整）。

输入样例：

10 a

输出样例：

aaaaaaaaaa
a        a
a        a
a        a
aaaaaaaaaa

思路：

题目相比与1027 打印沙漏难度不高，C语言入门题：

在四舍五入的写法上，参考了一些代码

于是代码如下：

#include<iostream>

using namespace std;

int main() {
    int n;
    char c;
    cin >> n >> c;
    int row = (n+n%2)/2;
    for(int i=0;i<row;i++){
        cout << c;
        if(i==0 || i==row-1) for(int j=0;j<n-2;j++) cout << c;
        else for(int j=0;j<n-2;j++) cout << ' ';
        cout << c << endl;
    }
    return 0;
}
