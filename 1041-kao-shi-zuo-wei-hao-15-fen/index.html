1041 考试座位号 (15分)

· 2020-05-08 · # 代码练习

题目

每个 PAT 考生在参加考试时都会被分配两个座位号，一个是试机座位，一个是考试座位。正常情况下，考生在入场时先得到试机座位号码，入座进入试机状态后，系统会显示该考生的考试座位号码，考试时考生需要换到考试座位就座。但有些考生迟到了，试机已经结束，他们只能拿着领到的试机座位号码求助于你，从后台查出他们的考试座位号码。

输入格式：

输入第一行给出一个正整数 N（≤1000），随后 N 行，每行给出一个考生的信息：准考证号 试机座位号 考试座位号。其中准考证号由 16 位数字组成，座位从 1 到 N 编号。输入保证每个人的准考证号都不同，并且任何时候都不会把两个人分配到同一个座位上。

考生信息之后，给出一个正整数 M（≤N），随后一行中给出 M 个待查询的试机座位号码，以空格分隔。

输出格式：

对应每个需要查询的试机座位号码，在一行中输出对应考生的准考证号和考试座位号码，中间用 1 个空格分隔。

输入样例：

4
3310120150912233 2 4
3310120150912119 4 1
3310120150912126 1 3
3310120150912002 3 2
2
3 4

输出样例：

3310120150912002 2
3310120150912119 1

思路：

题目又是数据查找类型的：

记录中优先考虑试机号作为特征码，因为题目最后是按照试机号进行查询

通过两个数组分别存储试机号对应的考生号与考试号

最后通过试机号直接进行访问输出即可

数组进行数据查询是十分方便的，如果需要对数据排序后再查询的话，数组可能就不怎么香了（因为数组排序后下标与数据不再是对应关系，需要进行额外处理），这种情况可以考虑使用map结构

于是代码如下：

#include<iostream>
#include<vector>

using namespace std;
int main() {
    int n;
    cin >> n;
    vector<string> v(n+1);
    vector<int> seat(n+1);
    for(int i=0;i<n;i++){
        string sno;int test,exam;
        cin >> sno >> test >> exam;
        v[test] = sno;
        seat[test] = exam;
    }
    cin >> n;
    for(int i=0;i<n;i++){
        int test;cin >> test;
        cout << v[test] << ' ' << seat[test] << endl;
    }
    return 0;
}
