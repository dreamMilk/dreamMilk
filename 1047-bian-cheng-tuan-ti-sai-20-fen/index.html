1047 编程团体赛 (20分)

· 2020-05-14 · # 代码练习

题目

编程团体赛的规则为：每个参赛队由若干队员组成；所有队员独立比赛；参赛队的成绩为所有队员的成绩和；成绩最高的队获胜。

现给定所有队员的比赛成绩，请你编写程序找出冠军队。

输入格式：

输入第一行给出一个正整数 N（≤10410^4104），即所有参赛队员总数。随后 N 行，每行给出一位队员的成绩，格式为：队伍编号-队员编号 成绩，其中队伍编号为 1 到 1000 的正整数，队员编号为 1 到 10 的正整数，成绩为 0 到 100 的整数。

输出格式：

在一行中输出冠军队的编号和总成绩，其间以一个空格分隔。注意：题目保证冠军队是唯一的。

输入样例：

6
3-10 99
11-5 87
102-1 0
102-3 100
11-9 89
3-2 61

输出样例：

11 176

思路：

题目与1032 挖掘机技术哪家强基本没有差别：

使用数组存储每个队伍的成绩（不知道这个队员编号是干嘛的，完全没有存储价值）

最后遍历数组找出最大值，并记录下标，然后输出即可

有一个输入时的方法：

在%后加上*可以忽略该字符的输入

于是代码如下：

#include<iostream>

using namespace std;

int main() {
    int n;
    cin >> n;
    int result[1001]={0};
    for(int i=0;i<n;i++){
        int team,score;
        scanf("%d-%*d %d",&team,&score);
        result[team] += score;
    }
    int champion=0,team=0;
    for(int i=1;i<1001;i++){
        if(result[i]>champion){champion = result[i]; team = i;}
    }
    cout << team << ' ' <<champion;
    return 0;
}
