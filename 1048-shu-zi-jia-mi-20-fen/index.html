1048 数字加密 (20分)

作者: Milk 's Home 日期: 2020-05-15 阅读时间:2.2 分钟 字数:528 分类: 代码练习 阅读量: -

题目

本题要求实现一种数字加密方法。首先固定一个加密用正整数 A，对任一正整数 B，将其每 1 位数字与 A 的对应位置上的数字进行以下运算：对奇数位，对应位的数字相加后对 13 取余——这里用 J 代表 10、Q 代表 11、K 代表 12；对偶数位，用 B 的数字减去 A 的数字，若结果为负数，则再加 10。这里令个位为第 1 位。

输入格式：

输入在一行中依次给出 A 和 B，均为不超过 100 位的正整数，其间以空格分隔。

输出格式：

在一行中输出加密后的结果。

输入样例：

1234567 368782971

输出样例：

3695Q8118

思路：

题目是字符串处理问题：

因为字符串的末尾是第一位，可以先反转字符再处理，或处理时从后往前

如果两个字符串的长度不一致，在较短字符串前填零，使长度一致

然后按照奇偶处理逻辑进行处理，并生成一个新的字符串后输出

在写代码过程中的注意点：

题目没说明，但是对于较长字符串多余的部分需要与'0'字符进行加密，而不是保持原形

因此字符串长短不一致时，不能通过截断较长字符串后与新字符串进行拼接（因为奇偶处理逻辑不同，且字符与'0'处理后并非原字符）

于是代码如下：

#include<iostream>

using namespace std;

int main() {
    string a,b,result;
    cin >> a >> b;
    bool flag =true;
    if(a.length()>b.length()) b.insert(0,a.length()-b.length(),'0');
    else a.insert(0,b.length()-a.length(),'0');
    for(int i=a.length()-1;i>=0;i--){
        if(flag){
            int temp = ((a[i]-'0')+(b[i]-'0'))%13;
            switch (temp){
                case 10: result.insert(0,1,'J');break;
                case 11: result.insert(0,1,'Q');break;
                case 12: result.insert(0,1,'K');break;
                default: result.insert(0,1,temp+'0');break;
            }
            flag=false;
        }else{
            int temp = b[i]-a[i];
            if(temp<0) result.insert(0,1,temp+10+'0');
            else result.insert(0,1,temp+'0');
            flag = true;
        }
    }
    cout << result;
    return 0;
}
