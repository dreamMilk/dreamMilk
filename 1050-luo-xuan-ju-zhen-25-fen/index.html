1050 螺旋矩阵 (25分)

2020-05-17 Views 代码练习718字4 min read繁

题目

本题要求将给定的 N 个正整数按非递增的顺序，填入“螺旋矩阵”。所谓“螺旋矩阵”，是指从左上角第 1 个格子开始，按顺时针螺旋方向填充。要求矩阵的规模为 m 行 n 列，满足条件：m×n 等于 N；m≥n；且 m−n 取所有可能值中的最小值。

输入格式：

输入在第 1 行中给出一个正整数 N，第 2 行给出 N 个待填充的正整数。所有数字不超过 104，相邻数字以空格分隔。

输出格式：

输出螺旋矩阵。每行 n 个数字，共 m 行。相邻数字以 1 个空格分隔，行末不得有多余空格。

输入样例：

12
37 76 20 98 76 42 53 95 60 81 58 93

输出样例：

98 95 93
42 37 81
53 20 76
58 60 76

思路：

首先整理一下题目的思路：

先从N\sqrt{N}N​开始往前寻找到列数n,然后确定行数m

将输入的数字按从大到小排序

因为每次螺旋一圈会减少两行，所以可以通过行数m确定循环次数（旋转圈数）

每次旋转一圈需要经过上、右、下、左四个边，每个边写成一个循环，将对应数字按序装入二维数组

最后将二维数组输出即可

需要注意的地方：

当第一次发现符合条件的n时，需要退出循环，避免获取到错误的n

当行数是偶数时，循环次数为行数/2，奇数时，循环次数为行数/2+1

当四条边循环时，需要注意每条边之间的重合部分，避免发生错误

只要在存储到二维数组的过程中仔细地对应下标数字，题目就没什么问题了

于是代码如下：

#include<iostream>
#include<vector>
#include<cmath>
#include<algorithm>

using namespace std;

int cmp(int a,int b){ return a>b; }

int main() {
    int N,m,n;
    cin >> N;
    for(int i=sqrt((double)N);i>0;i--){
        if(N%i == 0) {n=i; m=N/n; break;}
    }
    vector<int> v(N);
    for(int i=0;i<N;i++) scanf("%d",&v[i]);
    sort(v.begin(),v.end(),cmp);
    vector<vector<int>> V(m,vector<int>(n));
    int t=0,level = m%2+m/2;
    for(int i=0;i<level;i++){
        for(int j=i;j<n-i && t < N;j++) V[i][j] = v[t++];
        for(int j=i+1;j<m-i && t < N;j++) V[j][n-i-1] = v[t++];
        for(int j=i+1;j<n-i && t < N;j++) V[m-1-i][n-1-j] = v[t++];
        for(int j=i+1;j<m-1-i && t < N;j++) V[m-1-j][i] = v[t++];
    }
    for(int i=0;i<m;i++){
        for(int j=0;j<n;j++){
            if(j==0) cout << V[i][j];
            else cout << ' ' << V[i][j];
        }
        cout << endl;
    }
    return 0;
}
