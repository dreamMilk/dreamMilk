1052 卖个萌 (20分)

题目

萌萌哒表情符号通常由“手”、“眼”、“口”三个主要部分组成。简单起见，我们假设一个表情符号是按下列格式输出的：

[左手]([左眼][口][右眼])[右手]

现给出可选用的符号集合，请你按用户的要求输出表情。

输入格式：

输入首先在前三行顺序对应给出手、眼、口的可选符号集。每个符号括在一对方括号 []内。题目保证每个集合都至少有一个符号，并不超过 10 个符号；每个符号包含 1 到 4 个非空字符。

之后一行给出一个正整数 K，为用户请求的个数。随后 K 行，每行给出一个用户的符号选择，顺序为左手、左眼、口、右眼、右手——这里只给出符号在相应集合中的序号（从 1 开始），数字间以空格分隔。

输出格式：

对每个用户请求，在一行中输出生成的表情。若用户选择的序号不存在，则输出 Are you kidding me? @\/@。

输入样例：

[╮][╭][o][~\][/~]  [<][>]
 [╯][╰][^][-][=][>][<][@][⊙]
[Д][▽][_][ε][^]  ...
4
1 1 2 2 2
6 8 1 5 5
3 3 4 3 3
2 10 3 9 3

输出样例：

╮(╯▽╰)╭
<(@Д=)/~
o(^ε^)o
Are you kidding me? @\/@

思路：

题目难点在于如何获取每个[ ]内的表情：

使用第一层循环确定左括号，使用第二层循环确定右括号，然后使用substr函数截取

存放好每个表情后，判断用户输入的是否合理，如果合理，输出每个对应的表情

关于代码中的注意事项：

在表情输入时因为存在未知空格，需要使用getline函数进行整行读取，这也是为什么需要两层循环的原因

在输出转义字符\时需要在前面额外添加一个转义字符

于是代码如下：

#include<iostream>
#include<vector>

using namespace std;

int extract(vector<string> &v){
    string sign;
    getline(cin,sign);
    for(int i=0;i<sign.length()-1;i++){
        if(sign[i]=='['){
            for(int j=i+1;j<sign.length();j++){
                if(sign[j]==']'){
                    v.push_back(sign.substr(i+1,j-i-1));
                    break;
                }
            }
        }
    }
    return 1;
}

int main() {
    vector<string> hand,eye,mouth;
    extract(hand);
    extract(eye);
    extract(mouth);
    int k;
    cin >> k;
    for(int i=0;i<k;i++){
        int h1,e1,m,e2,h2;
        cin >> h1 >> e1 >> m >> e2 >> h2;
        if(h1>hand.size()||e1>eye.size()||m>mouth.size()||e2>eye.size()||h2>hand.size()||h1<1||e1<1||m<1||e2<1||h2<1){
            cout << "Are you kidding me? @\\/@" << endl;
        }
        else cout << hand[h1-1] << "(" << eye[e1-1] << mouth[m-1] << eye[e2-1] << ")" << hand[h2-1] << endl;
    }
    return 0;
}
