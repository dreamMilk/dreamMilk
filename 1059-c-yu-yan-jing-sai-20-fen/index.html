1059 C语言竞赛 (20分)

· 2020-05-26 · # 代码练习

题目

C 语言竞赛是浙江大学计算机学院主持的一个欢乐的竞赛。既然竞赛主旨是为了好玩，颁奖规则也就制定得很滑稽：

  • 0、冠军将赢得一份“神秘大奖”（比如很巨大的一本学生研究论文集……）。
  • 1、排名为素数的学生将赢得最好的奖品 —— 小黄人玩偶！
  • 2、其他人将得到巧克力。

给定比赛的最终排名以及一系列参赛者的 ID，你要给出这些参赛者应该获得的奖品。

输入格式：

输入第一行给出一个正整数 N（≤10410^4104），是参赛者人数。随后 N 行给出最终排名，每行按排名顺序给出一位参赛者的 ID（4 位数字组成）。接下来给出一个正整数 K 以及 K 个需要查询的 ID。

输出格式：

对每个要查询的 ID，在一行中输出 ID: 奖品，其中奖品或者是 Mystery Award（神秘大奖）、或者是 Minion（小黄人）、或者是 Chocolate（巧克力）。如果所查 ID 根本不在排名里，打印 Are you kidding?（耍我呢？）。如果该 ID 已经查过了（即奖品已经领过了），打印 ID: Checked（不能多吃多占）。

输入样例：

6
1111
6666
8888
1234
5555
0001
6
8888
0001
1111
2222
8888
2222

输出样例：

8888: Minion
0001: Chocolate
1111: Mystery Award
2222: Are you kidding?
8888: Checked
2222: Are you kidding?

思路：

题目比较容易：

可以使用一个数组将对应ID的名次进行存储，对于没有ID的名次置为0

在输出时，进行判断名次是否为0，为0说明查无此人

其他的名次按照题目条件进行判断

对于每一个查过的ID，需要将名次置为-1，避免重复查询

至于素数的判断可以参考直接1013 数素数

于是代码如下：

#include <iostream>

using namespace std;

bool judge(int i){
    if(i <= 3) return i > 1;
    if(i%6 != 1 && i%6 != 5) return false;
    for(int j=2;j*j <= i;j++){
        if(i%j == 0) return false;
    }
    return true;
}

int main() {
    int n;
    cin >> n;
    int ids[10000]={0};
    for(int i=1;i<=n;i++){
        int id;
        cin >> id;
        ids[id] = i;
    }
    int k;
    cin >> k;
    for(int i=0;i<k;i++){
        string id;
        cin >> id;
        if(ids[stoi(id)]==0) cout << id  << ": Are you kidding?" << endl;
        else if(ids[stoi(id)]==-1) cout << id << ": Checked" << endl;
        else{
            if(ids[stoi(id)]==1){
                cout << id << ": Mystery Award" << endl;
                ids[stoi(id)]=-1;
            }
            else if(judge(ids[stoi(id)])){
                cout << id << ": Minion" << endl;
                ids[stoi(id)]=-1;
            }
            else{
                cout << id << ": Chocolate" << endl;
                ids[stoi(id)]=-1;
            }
        }
    }
    return 0;
}
