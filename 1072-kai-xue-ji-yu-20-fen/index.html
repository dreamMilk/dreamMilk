1072 开学寄语 (20分)

· 2020-06-08 · # 代码练习

题目

下图是上海某校的新学期开学寄语：天将降大任于斯人也，必先删其微博，卸其 QQ，封其电脑，夺其手机，收其 ipad，断其 wifi，使其百无聊赖，然后，净面、理发、整衣，然后思过、读书、锻炼、明智、开悟、精进。而后必成大器也！

本题要求你写个程序帮助这所学校的老师检查所有学生的物品，以助其成大器。

输入格式：

输入第一行给出两个正整数 N（≤ 1000）和 M（≤ 6），分别是学生人数和需要被查缴的物品种类数。第二行给出 M 个需要被查缴的物品编号，其中编号为 4 位数字。随后 N 行，每行给出一位学生的姓名缩写（由 1-4 个大写英文字母组成）、个人物品数量 K（0 ≤ K ≤ 10）、以及 K 个物品的编号。

输出格式：

顺次检查每个学生携带的物品，如果有需要被查缴的物品存在，则按以下格式输出该生的信息和其需要被查缴的物品的信息（注意行末不得有多余空格）：

姓名缩写: 物品编号1 物品编号2 ……

最后一行输出存在问题的学生的总人数和被查缴物品的总数。

输入样例：

4 2
2333 6666
CYLL 3 1234 2345 3456
U 4 9966 6666 8888 6666
GG 2 2333 7777
JJ 3 0012 6666 2333

输出样例：

U: 6666 6666
GG: 2333
JJ: 6666 2333
3 5

思路：

题目比较容易：

将输入的违禁品编号使用bool ban[10000]进行存储，不属于违禁品的默认false

循环遍历学生以及携带物品，使用flag判断学生是否携带违禁品，并利用stu统计违规学生，cnt统计违禁品数量

flag还可以控制是否输出过名字，保证输出格式规范

对于违禁品编号的存储：

一开始发现题目中说明违禁品数量不超过6个，于是决定选择使用map存储

但是运行的时候发现，使用map存储的耗时要比数组慢很多，但是空间使用却没有太大的区别

数组与map相比，还是数组香，因此在相同情况下还是选择使用数组会比较好

于是代码如下：

#include<iostream>

using namespace std;

int main() {
    int n,m;
    scanf("%d%d",&n,&m);
    bool ban[10000]={false};
    for(int i=0;i<m;i++){
        int id;
        scanf("%d",&id);
        ban[id]=true;
    }
    int cnt=0,stu=0;
    for(int i=0;i<n;i++){
        char name[10];
        int num;
        bool flag=true;
        scanf("%s %d",&name,&num);
        for(int j=0;j<num;j++){
            int id;
            scanf("%d",&id);
            if(ban[id] && flag){
                printf("%s: %04d",name,id);
                cnt++;
                flag = false;
            }else if(ban[id]){
                printf(" %04d",id);
                cnt++;
            }
        }
        if(!flag){
            stu++;
            printf("\n");
        }
    }
    printf("%d %d\n",stu,cnt);
    return 0;
}
